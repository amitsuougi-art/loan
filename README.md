# LENS — 融資審査AI支援システム（デモ）

横浜銀行 融資部向け「融資審査AI支援システム LENS」の操作デモ（単一HTML）です。

## 機能
- ダッシュボード（審査タスク・リスクの集約）
- 類似案件検索（F-01）
- 稟議書ドラフト生成（F-02 / F-03、EYS連携・OCR数値転記）
- 事業性評価（F-04、行内格付・7観点スコア）
- 条件交渉支援（F-05）
- 稟議・承認ワークフロー（承認ルート・履歴管理）
- 与信リスクアラート（F-06 / F-07）

## 閲覧方法
`loan_demo.html` をブラウザ（Edge / Chrome）で開いてください。
※ デザイン部品（Tailwind / Chart.js / フォント）をCDNから読み込むため、閲覧時はインターネット接続が必要です。

## GitHub Pages（任意）
Settings → Pages で公開すると、ブラウザURLで閲覧・共有できます。
