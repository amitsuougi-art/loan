# LENS — 融資審査AI支援システム（デモ／社内配置用）

横浜銀行 融資部向け「融資審査AI支援システム LENS」の操作デモです。
**完全自己完結版**（外部CDN非依存）のため、インターネット接続がなくても動作します。

## 特長（社内配置向け）
- **単一HTMLファイル**で完結（Chart.js を内包、Webフォント不使用＝システムフォントで表示）
- **外部通信ゼロ**（閉域網・オフライン環境でもレイアウト崩れなし。検証済み：外部リクエスト0件）
- Webサーバー配置・共有フォルダ配置・ローカル閲覧のいずれでも動作

## 機能
- ダッシュボード（審査タスク・リスクの集約）
- 類似案件検索（F-01）
- 稟議書ドラフト生成（F-02 / F-03、EYS連携・OCR数値転記）
- 事業性評価（F-04、行内格付・7観点スコア）
- 条件交渉支援（F-05）
- 稟議・承認ワークフロー（承認ルート・履歴管理）
- 与信リスクアラート（F-06 / F-07）

## 配置方法
### A. 社内Webサーバー / イントラポータル
`loan_demo.html`（または `index.html`）を公開ディレクトリに置くだけ。
URLを開けば全員が閲覧でき、ダウンロード不要・常に最新を共有できます。
例：`http://（社内サーバー）/loan/` で `index.html` が開きます。

### B. 共有フォルダ（ファイルサーバー）
`\\サーバー名\共有\…` に `loan_demo.html` を配置。各自がダブルクリックでブラウザ表示。

### C. ローカル閲覧
ファイルをブラウザ（Edge / Chrome）にドラッグ＆ドロップ、またはダブルクリック。
